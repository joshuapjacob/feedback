'FEEBACK' was made for my final project of my introduction to web programming course (CSE104 @ Ã‰cole Polytechnique). It was inspired by Abelton Live. Just like any other musical playground online, this is nothing but a toy. I doubt you can create the next smash hit with this, but you're certainly welcome to prove me wrong!

The main audio framework was built using Tone.js, the fonts used are Open Sans and Gugi, the icons are from Font Awesome 5, and the fancy flying lines you see in the background are just a barely editied demo of AnimatedMeshLines which uses GSAP and is built on top of THREE.MeshLine. The neat little scroll bar is simplebar.
